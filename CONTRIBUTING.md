# Contributing

Thanks for your interest. Short version:

1. Fork, branch, open a PR.
2. Run `npm run lint && npm run test:coverage` locally before pushing.
3. Follow [Conventional Commits](https://www.conventionalcommits.org/) — CI and the changelog rely on it.

Full details — dev setup, commit conventions, review process, release process, testing philosophy — live in [`docs/contributor-guide.md`](docs/contributor-guide.md).

## Reporting bugs / requesting features

Use the issue templates under the "Issues" tab. For security issues, see [`SECURITY.md`](SECURITY.md).

## Code of conduct

All interaction is governed by our [Code of Conduct](CODE_OF_CONDUCT.md). Be kind; assume good faith.
