# Security Policy

## Reporting a vulnerability

Please report security issues privately via [GitHub Security Advisories](https://github.com/shashikanth-gs/a2a-mcp-skillmap/security/advisories/new). Do not open public issues for suspected vulnerabilities.

We aim to:

- Acknowledge your report within **1 business day**.
- Ship a patch release within **7 days** of confirming high-severity issues.

## Supported versions

The latest minor release of the current major line receives security fixes. Older majors are unsupported.

| Version | Supported |
|---|---|
| 0.1.x | ✓ |

## Scope + design

See [`docs/security.md`](docs/security.md) for the threat model, secret-handling rules, authentication scope, and input-validation guarantees.
